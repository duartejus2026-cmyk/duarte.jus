# Duarte Advocatus - Portal Jurídico de Elite

Este projeto é um portal jurídico moderno e profissional para o escritório **Duarte Advocatus**, focado em oferecer uma experiência premium para clientes que buscam excelência em Direito Médico, Bancário e Trabalhista.

## Tecnologias Utilizadas

- **React 19**: Biblioteca para construção da interface.
- **Vite**: Build tool extremamente rápida.
- **Tailwind CSS 4**: Framework CSS para design responsivo e moderno.
- **Framer Motion**: Biblioteca para micro-animações e transições suaves.
- **Lucide React**: Conjunto de ícones vetoriais.

## Estrutura do Projeto

```
src/
├── components/
│   ├── layout/       # Componentes globais (Header, Footer)
│   ├── sections/     # Seções da landing page (Hero, Specialties, etc.)
│   └── ui/           # Componentes de interface reutilizáveis
├── services/         # Integrações e lógica de negócio
├── types.ts          # Definições de tipos TypeScript
└── index.css         # Estilos globais e tokens de design
```

## Como Executar Localmente

1. **Instale as dependências:**
   ```bash
   npm install
   ```

2. **Configure as variáveis de ambiente:**
   Crie um arquivo `.env` baseado no `.env.example` (se houver integrações externas).

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

## Design & Experiência

O projeto segue padrões de design boutique, com paleta de cores sóbria (Deep Blue e Gold), tipografia serifada para autoridade e animações que conferem um ar tecnológico e premium.
